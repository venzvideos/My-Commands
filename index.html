What is Flutter & Why Use It?

Flutter is Google's open-source UI software development kit. It allows you to build beautiful, natively compiled applications for Mobile (iOS & Android), Web, and Desktop environments using only a single codebase.

Instead of writing separate Swift code for Apple and Java/Kotlin code for Android, you write your app UI layer just once using the Dart programming language.

Your current layout contains two critical developer commands:

1. cd [path]: Moves your console window directly into your dedicated workspace folder.

2. flutter run: Starts the compiler, fires up hot-reload, and boots the project live onto an attached device or emulator screen.

What is JSX & React?

React is a popular open-source JavaScript library developed by Meta (Facebook) used for building interactive User Interfaces, specifically for single-page web applications.

JSX (JavaScript XML) is a syntax extension used directly inside React. It allows developers to write structural HTML-like elements inside JavaScript files natively, making components readable and fast to compile.

Your workspace controls include:

1. npm start: Fires up your Node package runtime and starts a local web server (usually localhost:3000) for instant programming updates.

2. npm run build: Compresses, cleans, and builds your code assets into high-performance plain files ready to deploy onto a server layout.
